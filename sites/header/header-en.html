The header version one (v1) is deprecated but still supported.

Version 2

This version requires the implementation of the gcweb-menu version 2 where the <div class="container"> is relocated in the header so outside the gcweb-menu component.

Default header

<header>
	<div id="wb-bnr" class="container">
		<div class="row">
			<!-- Language toggle [version 1] -->
			<!-- Branding [version 1] -->
			<!-- Search [version 1] -->
		</div>
	</div>
	<hr>
	<div class="container">
		<div class="row">
			<div class="col-sm-12">
				<!-- Site Menu [version 2] -->
			</div>
		</div>
	</div>
	<!-- Breadcrumbs [version 1] -->
</header>
