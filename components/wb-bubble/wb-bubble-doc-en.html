Technical name
  • wb-bubble

Purpose

Quick action bubble located at the bottom-right corner of a page that open an overlay in the context of a virtual assistant or action a link to a page.

Usage

Use when

To send an action to a wet-boew bubble plugin.

Do not use when

When executing a such action can make your page not WCAG compliant.

Working examples

  • English: Quick action bubble with default avatar - opening a link
  • French: Bulle liée à une action rapide avec l'avatar par défaut - overture d'un lien
  • English: Virtual assistant bubble with customized avatar - opening an overlay
  • French: Bulle d'assistant virtuel avec un avatar personnalisé - overture d'une fenêtre superposée

How to implement

  • Add the wb-bubble class to a <button> or a <a> element in order to apply the bubble pattern.
  • When using the <button> element, add and set the data-wb-doaction attribute in order to open the overlay wet pluggin.
  • When using the link <a> element, set the hrefattribute to the page as direct link..

Template (v1.0)

Notes:

  • Can only be used with a button (<button>) to open an overlay or with a link element (<a>) to open a page.

Button

	<button data-wb-doaction='{ "action": "openDialog", "source": "#mid-screen" }' class="wb-bubble"></button>

Where:

"action": "openDialog"
Will open an overlay by using the overlay plugin.
"source": #mid-screen
The source represent a pointer to section id, which will be included in the overlay panel

Hyperlink

<a href="#" class="wb-bubble">Quick link</a>

Visual rendering (v1.0)

  • Round shape icon.
  • Drop shadow.
  • White background.
  • Red maple leaf avatar (default).
  • Black 1px border on focus
  • Bottom-right corner fix location.
  • Size of 60px x 60px.
  • Dedicated space of 30px from the right and bottom.
  • Notification panel:
    • Hidden by default
    • Open on the left of the bublle
    • White text on blue background (hex #335075)
    • Small font-size (.8em)
    • Close button "X" is bigger in mobile view
