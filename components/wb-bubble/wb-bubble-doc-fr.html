But

Bulle d'action rapide située dans le coin inférieur droit d'une page qui ouvre une superposition dans le contexte d'un assistant virtuel ou actionne un lien vers une page.

Utilisation

Utiliser quand

Pour envoyer une action au plugiciel de bulle wet-boew.

Ne pas utiliser quand

L'exécution d'une telle action peut rendre votre page non conforme aux normes d'accessibilités (WCAG).

Exemples pratiques

  • French: Bulle liée à une action rapide avec l'avatar par défaut - overture d'un lien
  • English: Quick action bubble with default avatar - opening a link
  • French: Bulle d'assistant virtuel avec un avatar personnalisé - overture d'une fenêtre superposée
  • English: Virtual assistant bubble with customized avatar - opening an overlay

Comment mettre en œuvre

  • Ajouter la classe wb-bubble à un élément <button> ou <a> afin d'appliquer l'aspect de bulle.
  • Lorsque l'élément boutton est utilisé <button>, ajoutez et configurez l'attribut data-wb-doaction afin d'actionner le plugiciel superposé de wet-boew.
  • Lorsque l'élément de lien <a> est utilisé, ajoutez l'URL de la page à l'attribut href.

Évaluation et rapport

Il n'y a pas d'évaluation et de rapport disponible pour ce composant.

Gabarit (v1.0)

Notes:

  • Ne peut être utilisé qu'avec un élément boutton (<button>) ou de lien (<a>).

Boutton

	<button  data-wb-doaction='{ "action": "openDialog", "source": "#mid-screen" }' class="wb-bubble"></button>

Ou:

"action": "openDialog"
Will open an overlay by using the overlay plugin.
"source": #mid-screen
The source represent a pointer to section id, which will be included in the overlay panel

Hyperlien

<a href="#" class="wb-bubble">Lien rapide</a>

Rendu visuel (v1.0)

  • Icône de forme ronde
  • Ombre portée
  • Arrière-plan blanc
  • Bordure noir de 1px lorsqu'elle obtient le focus
  • Feuille d'érable rouge comme avatar (par défaut)
  • Positionnement fixe dans le coin inférieur droit
  • Dimmensions de 60px x 60px
  • Espace dédiée de 30px du bas et de la droite
  • Panneau de notification:
    • Caché par défaut
    • Ouvre à la gauche de la bulle
    • Texte blanc sur arrière-plan bleu (hex #335075)
    • Taille de la police petite (.8em)
    • Boutton de fermeture "X" plus gros en mode mobile
