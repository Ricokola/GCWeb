Purpose

This feature provides a quiz pattern when applied to the Steps Form component.

How to use

  1. Add the quiz class to <div> element containing the wb-steps class of the Steps Form component.

Content limited width panel

  • Apply panel and panel-default classes to the first <section> element.
  • Add the panel-body class to the <div> element containing the wb-steps class.
